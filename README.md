## Strategy Rules

| Parameter | Value |
|-----------|-------|
| Asset | XAUUSD (Gold) |
| Timeframe | M5 |
| Opening Range | 9:30 AM – 9:35 AM New York |
| Entry | Instant (market at breakout candle close) |
| Breakout (Long) | M5 candle closes ABOVE OR High |
| Breakout (Short) | M5 candle closes BELOW OR Low |
| Stop Loss | Opposite side of OR ± 2 ticks |
| Take Profit | 5R |
| Wick Filter | ON — top/bottom wick ≤ 50% of candle body |
| Distance Filter | ON — breakout close within 0.1×–1.6× OR size |
| Second Chance | ON — one opposite breakout allowed after first trade closes |
| Force Close | 4:00 PM New York |
| Days | Monday – Friday |
